# Generated Go Stubs

This directory contains Go Protobuf stubs generated by `buf generate` using the
`buf.build/protocolbuffers/go` plugin (Gap 540).

These files are auto-generated â€” do not edit manually. Run `make generate` to regenerate.

## Usage

```go
import (
    eventsv1 "github.com/muveraai/aumos-proto/gen/go/aumos/events/v1"
    modelsv1 "github.com/muveraai/aumos-proto/gen/go/aumos/models/v1"
)

event := &eventsv1.ModelLifecycleEvent{
    EventId:  uuid.New().String(),
    TenantId: "tenant-123",
    ModelId:  "model-456",
    Action:   eventsv1.ModelLifecycleAction_MODEL_DEPLOYED,
}
```

## Regenerating

```bash
make generate
```

Requires `buf` to be installed: `brew install buf` or see https://buf.build/docs/installation
