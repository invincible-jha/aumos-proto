# Generated TypeScript Stubs

This directory contains TypeScript Protobuf stubs generated by `buf generate` using the
`buf.build/bufbuild/es` plugin (protobuf-es) (Gap 540).

These files are auto-generated â€” do not edit manually. Run `make generate` to regenerate.

## Usage

```typescript
import { ModelLifecycleEvent, ModelLifecycleAction } from "./aumos/events/v1/model_lifecycle_pb";

const event = new ModelLifecycleEvent({
  eventId: crypto.randomUUID(),
  tenantId: "tenant-123",
  modelId: "model-456",
  action: ModelLifecycleAction.MODEL_DEPLOYED,
});

// Serialize to binary
const bytes = event.toBinary();

// Deserialize from binary
const restored = ModelLifecycleEvent.fromBinary(bytes);
```

## Regenerating

```bash
make generate
```

Requires `buf` to be installed: `brew install buf` or see https://buf.build/docs/installation
